MOW-IT-NOW

Simulateur de tondeuses à gazon.  L'application est développée & testée sous windows et nécéssite maven.

COMPILATION
===========

mvn clean install

LANCEMENT
=========

mvn exec:java

Le programme attendra les instructions sur l'entrée standard.  Exemple de données d'entrée :
		
5 5
1 2 N
GAGAGAGAA
3 3 E
AADAADADDA
